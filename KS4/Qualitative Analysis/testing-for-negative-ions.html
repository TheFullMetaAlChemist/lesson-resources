Negative Ion Testing

Interactive GCSE Chemistry tool for identifying carbonate, halide and sulfate ions using standard chemical tests.

Negative Ion Identification Flowchart

Start with the unknown solution or solid sample

Carbonate Test

Add dilute acid

Effervescence / fizzing
+
Gas turns limewater cloudy
Carbonate ion, CO₃²⁻

Halide Test

Add dilute nitric acid, then silver nitrate solution

White precipitate → Chloride, Cl⁻
Cream precipitate → Bromide, Br⁻
Yellow precipitate → Iodide, I⁻

Sulfate Test

Add dilute hydrochloric acid, then barium chloride solution

White precipitate forms
Sulfate ion, SO₄²⁻
