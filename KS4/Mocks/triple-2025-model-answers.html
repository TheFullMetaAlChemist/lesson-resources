Question 1 – Atomic Structure & the Periodic Table

Model answers and common mistakes exactly as previously drafted.

Q1.1 Where are non-metals found in the periodic table?

1 mark

Model answer

Non-metals are found towards the top right-hand side of the periodic table.

Common mistakes

  • Saying only “on the right” (too vague)
  • Listing elements instead of describing position
  • Saying “Group 0” only

Q1.2 Which property of gallium is not typical of most metals?

1 mark

Model answer

A melting point below 100 °C.

Common mistakes

  • Choosing “conducts electricity” (typical of metals)
  • Choosing “forms positive ions” (true for metals)

Q1.3 Electronic structure of sodium

1 mark

Model answer

11 electrons arranged as 2, 8, 1.

Common mistakes

  • Using 2, 8, 2
  • Putting all electrons on one shell
  • Incorrect total number of electrons

Q1.4 Why did Mendeleev not place tellurium and iodine in order of atomic weight?

1 mark

Model answer

He placed elements so that those with similar properties were in the same group, even if this meant not following atomic weight order.

Common mistakes

  • Talking about electron shells (not known at the time)
  • Saying “he made a mistake”
  • Mentioning atomic number

Q1.5 Explain why argon is unreactive

2 marks

Model answer

Argon has a full outer shell of electrons, which gives it a stable electronic structure, so it does not gain, lose or share electrons.

Common mistakes

  • Saying “argon has no outer shell”
  • Saying “argon doesn’t react” with no explanation
  • Mentioning protons or neutrons

Q1.6 What is meant by “isotopes”?

2 marks

Model answer

Isotopes are atoms of the same element with the same number of protons but different numbers of neutrons.

Common mistakes

  • Saying “same number of electrons”
  • Not stating protons explicitly
  • Describing mass only

Q1.7 Estimate the relative atomic mass of X

Common mistakes

  • Choosing the smaller mass number
  • Trying to over-calculate when an estimate is expected

Q2.2 Describe what happens when calcium reacts with sulfur

4 marks

Model answer

A calcium atom loses two electrons and a sulfur atom gains two electrons. This transfer forms Ca²⁺ and S²⁻ ions, which attract to form an ionic compound.

Common mistakes

  • Saying electrons are shared (that’s covalent)
  • Naming only one ion
  • Missing the electron transfer of two electrons

Q2.3 Describe the structure and bonding of graphite

4 marks

Model answer (points to include)

Graphite is a giant covalent structure made of layers of hexagonal rings. Each carbon forms three covalent bonds, leaving delocalised electrons.

Common mistakes

  • Describing diamond instead
  • Saying there are covalent bonds between layers
  • Forgetting delocalised electrons

Q2.4 Why do small molecules not conduct electricity?

1 mark

Model answer

They do not contain charged particles that can move (no ions / no delocalised electrons).

Common mistakes

  • Talking about “not enough electrons” without mentioning movement/charge
  • Confusing small molecules with ionic lattices

Q2.5 Why does poly(ethene) have a higher melting point than methane?

1 mark

Model answer

Poly(ethene) has stronger intermolecular forces to overcome than methane.

Common mistakes

  • Saying “because it’s bigger” without linking to intermolecular forces
  • Talking about covalent bonds breaking (melting breaks intermolecular forces, not covalent bonds)

Q4.5 Explain the trend in reactivity down Group 7

4 marks

Model answer

Reactivity decreases down the group. Lower down the group atoms have more shells, so the outer shell is further from the nucleus and more shielded. This reduces attraction for an incoming electron, so it is harder to gain an electron.

Common mistakes

  • Saying reactivity increases down the group
  • Talking about losing electrons (halogens gain one)
  • Missing the shielding → weaker attraction link

Question 7 – Hydrazine, Bonding & Energetics

Q7.1 Evaluate using hydrazine instead of hydrogen in a fuel cell

4 marks

Model answer

Hydrazine gives a higher voltage (1.61 V) than hydrogen (1.23 V), so fewer cells may be needed. It is a liquid, so easier to store and transport than hydrogen gas. It also does not need a platinum catalyst, potentially reducing cost. However, hydrazine is toxic (hydrogen is explosive), so safety is a major concern. Overall: useful for voltage/storage and catalyst cost, but toxicity is a serious drawback.

Common mistakes

  • Listing facts with no comparison or judgement
  • No conclusion (must decide and justify)
  • Ignoring hazards information

Q7.4 Calculate the bond energy (X)

4 marks

Model answer (full working)

Bonds broken = (4×391) + 498 + X 4×391 = 1564 1564 + 498 = 2062 So broken = 2062 + X Bonds formed = 945 + (4×464) 4×464 = 1856 945 + 1856 = 2801 Given: formed is 581 kJ/mol greater than broken 581 = formed − broken 581 = 2801 − (2062 + X) 581 = 739 − X X = 739 − 581 = 158 kJ/mol

Answer: X = 158 kJ/mol

Common mistakes

  • Using the 581 difference the wrong way round
  • Missing brackets in 2801 − (2062 + X)
  • Arithmetic slip with 2801 − 2062

Q8.4 Calculate concentration of sodium hydroxide

4 marks

Model answer (full working)

Given: V(H2SO4) = 25.0 cm³ = 0.0250 dm³ c(H2SO4) = 0.216 mol/dm³ V(NaOH) = 11.25 cm³ = 0.01125 dm³ Equation: 2NaOH + H2SO4 → Na2SO4 + 2H2O 1) Moles of acid: n(H2SO4) = c×V = 0.216 × 0.0250 = 0.00540 mol 2) Mole ratio (NaOH : H2SO4 = 2 : 1): n(NaOH) = 2 × 0.00540 = 0.0108 mol 3) Concentration of NaOH: c(NaOH) = n/V = 0.0108 ÷ 0.01125 = 0.960 mol/dm³

Answer: 0.960 mol/dm³

Common mistakes

  • Forgetting to divide cm³ by 1000
  • Using the wrong mole ratio (forgetting the 2)
  • Using 11.25 as dm³ instead of 0.01125

Q8.5 Correct statement about HCl and HCN

1 mark

Model answer

HCl is a more dilute and stronger acid than HCN.

Common mistakes

  • Confusing strength with concentration
  • Assuming “more concentrated” always means lower pH

Q8.6 Explain why HCl has a lower pH than HCN using the data

2 marks

Model answer

HCl has a lower pH because it produces a much higher concentration of H⁺ ions. HCl is 0.1 mol/dm³ and 100% ionised so [H⁺] = 0.1. HCN is 2.0 mol/dm³ but only 0.01% ionised so [H⁺] = 2.0 × 0.0001 = 0.0002. Therefore HCl has far more H⁺ ions, so pH is lower.

Common mistakes

  • Saying “HCN is more concentrated so lower pH” (wrong)
  • Not using the ionisation data

Question 9 – Making a Salt + Percentage Yield

Model answers and common mistakes exactly as previously drafted.

Q9.1 Plan a method to obtain crystals of zinc iodide

6 marks

Model answer (method)

1) Dissolve iodine in ethanol (stir). 2) Add zinc a little at a time and stir until zinc is in excess (solid zinc remains). 3) Filter to remove excess zinc. 4) Gently evaporate some ethanol using a water bath/electric heater (ethanol is flammable – no naked flames). 5) Allow to cool and crystallise zinc iodide. 6) Filter crystals and dry (e.g., between filter paper).

Common mistakes

  • Heating ethanol with a Bunsen burner
  • No filtration step (product contaminated with zinc)
  • Not concentrating the solution before crystallising

Q9.2 Calculate the percentage yield

5 marks

Model answer (full working)

Given: mass I2 = 6.35 g mass ZnI2 produced = 6.29 g Mr(I2) = 254 Mr(ZnI2) = 319 Equation: Zn + I2 → ZnI2 (1:1) 1) Moles of I2: n(I2) = mass / Mr = 6.35 / 254 = 0.0250 mol 2) Moles of ZnI2 (1:1): n(ZnI2) = 0.0250 mol 3) Theoretical mass of ZnI2: m = n×Mr = 0.0250 × 319 = 7.975 g 4) % yield: % = (actual / theoretical) × 100 % = (6.29 / 7.975) × 100 = 78.871... ≈ 78.9%

Answer: 78.9% (3 s.f.)

Common mistakes

  • Inverting the yield fraction (theoretical/actual)
  • Rounding too early
  • Using 6.35 g as the product mass
