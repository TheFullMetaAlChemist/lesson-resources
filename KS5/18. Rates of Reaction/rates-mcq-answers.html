A Level OCR Chemistry • 5.1 Rates, Orders & Arrhenius

MCQ Answers & Explanations

A worked answer guide for the Rates, Orders and Arrhenius multiple-choice set. Use the tabs to move between Easy, Medium and Hard questions.

Systematic Approach

Use before checking

1. Read the rate equation

Species missing from the rate equation are zero order. [X] means first order and [X]2 means second order.

2. Compare changes carefully

Use factor changes in concentration and rate. Try to compare experiments where only one concentration changes.

3. Check units at the end

Use k = rate ÷ concentration terms. For Arrhenius questions, do not mix J and kJ.

Key Arrhenius relationship: ln k = (-Ea / R)(1/T) + ln A On a graph of ln k against 1/T, the gradient is -Ea/R and the y-intercept is ln A.

Easy Questions

5 questions

These focus on recognising order from equations, interpreting simple graphs and using basic Arrhenius units.

Easy Q1 Answer: C

First order rate–concentration graph

First order with respect to propanone means the rate is directly proportional to the concentration of propanone. If concentration doubles, rate doubles.

A first order rate–concentration graph is a straight line through the origin.
Easy Q2 Answer: B

Using orders to predict a new initial rate

The reaction is zero order with respect to W and first order with respect to X. W stays the same, so it has no effect here. X doubles from 0.50 to 1.00 mol dm-3.

new rate = 7.0 × 10-4 × 2 = 1.4 × 10-3 mol dm-3 s-1
Easy Q3 Answer: D

Units of the rate constant

The rate equation is:

rate = k[X][Y]

Overall order = 2, so:

k = mol dm-3 s-1 ÷ (mol dm-3)2

This simplifies to:

dm3 mol-1 s-1
Easy Q4 Answer: A Unit issue worth noting

Arrhenius units and activation energy

The intended answer is A, Ea, because activation energy is commonly quoted at A-level in kJ mol-1.

However, the key idea is that the units must match in the Arrhenius equation:

k = Ae-Ea/RT

The power of e, which is -Ea/RT, must have no units. If R is given as:

R = 8.31 J K-1 mol-1

then RT has units of:

J mol-1

So Ea must also be in J mol-1 when using this value of R. If Ea is quoted in kJ mol-1, R must be converted to:

0.00831 kJ K-1 mol-1
Student note: Activation energy can be written in either J mol-1 or kJ mol-1, but it must match the units used for R. Always check whether your final answer needs to be divided by 1000.
Easy Q5 Answer: A

Using an Arrhenius plot

The rearranged Arrhenius equation is:

ln k = (-Ea / R)(1/T) + ln A

This has the same form as y = mx + c. The gradient is -Ea/R, so Ea can be calculated from the gradient.

Medium Questions

5 questions

These questions bring together concentration–time graphs, rate–concentration graphs, rate equations and calculations of k.

Medium Q1 Answer: A

Zero order concentration–time graph

For a zero-order reaction, the rate is constant until the reactant runs out. This means the concentration decreases by the same amount every second.

Zero order concentration–time graph = straight downward line.
Medium Q2 Answer: C

Reading orders from the rate equation

The rate equation is:

rate = k[H+][C3H6O]

Iodine is not in the rate equation, so it is zero order with respect to iodine. Propanone is first order. Sulfuric acid provides H+, so the reaction is first order with respect to acid.

Medium Q3 Answer: C

Calculating k from initial rates

The rate equation is:

rate = k[A][B]2

Use experiment 2:

1.5 × 10-4 = k × 0.30 × (0.25)2 k = 1.5 × 10-4 ÷ (0.30 × 0.0625) k = 8.0 × 10-3
Medium Q4 Answer: D

Second order rate–concentration graph

Second order means:

rate ∝ [reactant]2

This produces an upward curve through the origin, because increasing concentration has a larger and larger effect on rate.

Medium Q5 Answer: A

Activation energy from gradient

For a graph of ln k against 1/T:

gradient = -Ea / R

The gradient is -45000, so:

Ea = -gradient × R Ea = 45000 × 8.31 = 373950 J mol-1 Ea = 374 kJ mol-1

Hard Questions

5 questions

These require more careful use of Arrhenius plots, rate equations, rate-determining steps and unit conversions.

Hard Q1 Answer: D

Pre-exponential factor from an Arrhenius plot

For an Arrhenius plot:

ln k = (-Ea / R)(1/T) + ln A

The y-intercept is ln A. From the graph, the intercept is approximately 31.5.

ln A = 31.5 A = e31.5 = 4.79 × 1013 s-1
Hard Q2 Answer: A

Rate constant from a third-order rate equation

The rate equation is:

rate = k[CH3COCH3][H+]2

Substitute the values:

2.75 × 10-3 = k × 3.32 × (0.73)2 k = 2.75 × 10-3 ÷ [3.32 × (0.73)2] k = 1.55 × 10-3

Overall order = 3, so the units of k are:

dm6 mol-2 s-1
Hard Q3 Answer: C intended Printed unit flaw

Activation energy from Arrhenius data

Use:

gradient = -Ea / R Ea = -gradient × R

Using the data, the gradient is approximately:

gradient ≈ -1.09 × 104

Therefore:

Ea = -(-1.09 × 104) × 8.31 Ea = 90700 J mol-1 Ea = 90.7 kJ mol-1
Student note: The intended option is C, but the printed unit is flawed if it says 90700 kJ mol-1. It should say either 90700 J mol-1 or 90.7 kJ mol-1. This is exactly why unit checking matters in Arrhenius questions.
Hard Q4 Answer: A

Finding the rate equation from experimental data

Compare experiments 1 and 3. A and B stay the same, while C halves from 0.42 to 0.21. The rate decreases from 3.2 × 10-3 to 8.0 × 10-4, which is a factor of 4 decrease.

If halving C quarters the rate, the reaction is second order with respect to C.

Experiments 1, 2 and 4 then show that A is first order and B is zero order. Therefore:

rate = k[A][C]2
Hard Q5 Answer: B

Mechanism and rate-determining step

The rate equation is:

rate = k[NO]2[H2]

Statement 1 is true. If [NO] is halved, the rate changes by:

(1/2)2 = 1/4

If [H2] is doubled, the rate changes by:

× 2

Overall effect:

1/4 × 2 = 1/2

Statement 2 is true. The second step can be the rate-determining step because the earlier step forms N2O2, allowing the rate equation to include [NO]2 and [H2].

Statement 3 is false. The rate constant changes with temperature. This is one of the key ideas shown by the Arrhenius equation.

True statements: 1 and 2 only.
